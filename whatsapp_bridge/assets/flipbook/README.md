# Canopy Hills Villas Flipbook

Open `index.html` in a browser to view the interactive client book.

To publish online, upload the whole `output/flipbook` folder to hosting. Keep the folder structure unchanged:

- `index.html`
- `assets/webp/`
- `assets/Canopy_Hills_Villas_Book.pdf`

The WebP pages are optimized for web viewing. The original PDF is included as a download link.
